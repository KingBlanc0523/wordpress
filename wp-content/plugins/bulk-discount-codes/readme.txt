Bulk Discount Codes add-on allows you to create large number of discounts codes at once.

== Description ==

Bulk Discount Codes add-on allows you to create large number of discounts codes at once.
That way you can create, for instance, 1000 discount codes easily and save your time. You just need to copy and paste new-line-separated list of discount codes and the add-on will do the hard part for you.
